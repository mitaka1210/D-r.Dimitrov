## Getting Started

To be able to start the project, clone the repository or download it.

### Prerequisites

- npm
  ```sh
  npm install npm@latest -g
  ```

### Installation

1. Clone the repo
   ```sh
   git clone git@github.com:mitaka1210/d-r.Dimitrov.git
   ```
2. Install NPM packages
   ```sh
   npm install
   ```
